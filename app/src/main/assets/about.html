ABOUT

1. Pengguna dapat mengunduh data titik panas di laman SiPongi dalam bentuk TXT kemudian ubah ekstensinya menjadi CSV.

2. Pastikan akses storage pada perangkat pengguna dalam posisi enabled.

3. Buat folder pada perangkat dengan nama dbSimpantas, lalu simpan fail titik panas ke dalam folder ini.

4. Format penamaan fail harus sesuai dengan nama bulan dan tahun. Contohnya jika data titik panas yang diolah adalah data bulan Januari 2023, maka penamaan fail harus JANUARI2023.csv

5. Untuk fitur SPADE disarankan menggunakan data bulanan, sedangkan untuk fitur DBSCAN disarankan menggunakan data harian.

1. Nilai minimum support harus berada di rentang 0 hingga 1.

2. Nilai minimum points harus berupa bilangan bulat positif lebih atau sama dengan 1.
